 1) 분수 추상 자료형(ADP)
- 분자(num)와 분모(den)를 정수형 변수로 가집니다.
- 연산 (Operation): 
  1. 더하기(+), 빼기(-), 곱하기(*), 나누기(/) 기능을 수행합니다.
  2. 계산 결과는 최대공약수(GCD)를 이용해 기약분수 형태로 만듭니다.
  3. "A/B" 모양의 글자를 분수 데이터로 변환합니다.

 2) GCD, LCM 알고리즘 
- GCD (최대공약수): 
  두 수 A, B가 있을 때 A를 B로 나눈 나머지를 구합니다. 나머지가 0이 될 때까지 A에 B를 넣고, B에 나머지를 넣는 과정을 반복합니다. 마지막에 남은 A가 최대공약수입니다.
- LCM (최소공배수): 
  (A * B)를 두 수의 최대공약수(GCD)로 나누면 최소공배수가 됩니다
